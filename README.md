# Basic 850

Mobile-first Basic English 850 vocabulary web app.

## Features

- 850 Basic English words by category
- Word and example sentence pronunciation
- Five example sentence patterns per word
- Tap an example sentence to reveal Chinese
- Local progress tracking
- JSON progress export/import backup

## Deploy to GitHub Pages

This is a static site. Upload these files to a public GitHub repository:

- `index.html`
- `.nojekyll`
- `src/app.js`
- `src/styles.css`
- `src/wordData.js`

Then enable GitHub Pages from the repository root.
