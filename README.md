# Basic 850 英语学习网页

一个面向手机使用的 Basic English 850 单词学习网页。项目最初目标是做一个可以直接在 iPhone 上打开、随时学习基础英语词汇的小工具，后续围绕移动端体验、发音、例句、进度备份和学习反馈做了多轮迭代。

在线地址：

https://reed457216746-coder.github.io/basic-850/

## 核心功能

- **850 个 Basic English 基础词汇**  
  按操作词、一般事物、可见事物、性质、反义性质等类别组织，适合做基础词汇积累。

- **适配 iPhone 的移动端界面**  
  页面按手机优先设计，底部导航固定，学习卡片、复习、测试和我的页面都适合单手使用。

- **单词朗读和例句朗读**  
  使用浏览器语音能力播放单词和例句，支持美音、英音切换。iPhone 上会使用系统可用的英文语音。

- **完整音标覆盖**  
  已为 850 个单词补齐 IPA 音标，并修复了早期出现的乱码和问号音标问题。

- **每个单词 5 条应用例句**  
  例句包含基础理解、生活场景、问句练习、常见搭配、对比感知。点击例句可显示中文翻译，也可以单独朗读例句。

- **学习状态记录**  
  支持“掌握”和“不熟”两种状态。点击“掌握”或“不熟”后自动进入下一个词。

- **复习队列**  
  标记为“不熟”的词会进入复习页，方便集中回看。

- **听音选词测试**  
  根据中文释义在多个英文选项中选择正确单词，用来巩固记忆。

- **进度导入导出**  
  学习进度保存在浏览器本地，同时支持导出 JSON 文件和导入恢复，降低清理浏览器或换设备造成进度丢失的风险。

- **学习正反馈音效**  
  支持掌握、不熟、例句展开、测试答对、测试答错、每掌握 5 个词、每掌握 15 个词等音效。音效开关在“我的”页面中。

## 进度保存在哪里

学习进度默认保存在浏览器的 `localStorage` 中，包含：

- 已掌握单词
- 不熟单词
- 美音 / 英音偏好
- 学习音效开关

如果用户清理 Safari 网站数据、更换浏览器或更换设备，浏览器本地进度可能丢失。因此建议定期使用“我的 -> 导出进度”保存 JSON 备份文件。

## 技术说明

这是一个纯静态网页项目，不需要后端服务。

主要文件：

- `index.html`：页面入口
- `src/app.js`：应用逻辑、学习状态、朗读、音效、交互
- `src/styles.css`：移动端界面样式
- `src/wordData.js`：850 单词、中文释义、音标数据
- `assets/sounds/`：自定义学习音效

## 部署方式

项目部署在 GitHub Pages 上。仓库开启 Pages 后，直接从仓库根目录发布即可。

部署需要包含：

- `index.html`
- `.nojekyll`
- `src/`
- `assets/`
- `README.md`
